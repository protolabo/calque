Description

Poursuivant le travail réalisé cet été dans le projet Calque, nous souhaitons enrichir l'outil cet automne pour permettre la création de cartes multicouches. Ces couches offriront la possibilité de modéliser des étages ou d'abstraire des informations, permettant ainsi une visualisation plus complexe et organisée. De plus, nous allons améliorer les fonctionnalités d'annotation et de catégorisation, en introduisant notamment le concept de ligne, qui permet de traiter un ensemble d'arêtes de manière cohérente.
